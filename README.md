# AstrBot 高情商输出卫兵 (LLM Output Guard)

本插件为 AstrBot 提供了一个强大的输出过滤与质量控制方案，防止 AI 在群聊中产生长篇大论、胡言乱语或泄露思考过程。

## 核心特性

- **思考过程切除**：自动识别并彻底移除 `<think>` 标签内容。
- **字数熔断机制**：支持自定义字数阈值，超过限制即刻拦截。
- **双模型质量校验**：利用监工模型判定输出质量，自动区分“详细解答”与“无意义碎碎念”。
- **高情商自动重写**：对于判定不合格的长文，自动重写为简短、得体的回复。
- **场景化配置**：群聊与私聊独立开关，支持群聊黑名单。

## 安装方法

1. 在 AstrBot 插件管理中搜索 `llm_output_guard` 或通过本仓库 URL 安装。
2. 重启 AstrBot。

## 配置项说明

| 配置项 | 说明 | 默认值 |
| :--- | :--- | :--- |
| `limit` | 触发校验的字数阈值。 | `100` |
| `enable_group` | 是否在群聊中启用。 | `true` |
| `enable_private` | 是否在私聊中启用。 | `false` |
| `monitor_prompt` | 监工判定的 Prompt 规则。 | `内置默认模板` |
| `monitor_model` | 指定监工模型（可选）。 | `默认聊天模型` |
